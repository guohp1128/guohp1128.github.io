一，hugo博客搭建

1，安装和配置git

2，安装hugo和添加环境变量

3，查看hugo版本

4，git进入hugo目录

5，创建新的hugo网址

​ hugo new site blog

6，git进入blog目录

7，下载theme

​ git clone https://github.com/guohp1128/gohugo-theme-ananke.git themes/ananke

8，配置主题名称

​ theme=‘ananke’

9，启动博客在本地查看博客内容

​ hugo server

10，部署在github上，用 hugo 生成网页，托管到 GitHub 仓库

​ hugo –theme=ananke –baseURL=“https://guohp1128.github.io/"

11，将 blog/public 目录 push 到刚创建仓库的 master 分支

cd public git init git add . git commit -m “message” git remote add origin https://github.com/guohp1128/guohp1128.github.io.git git push -u origin master
